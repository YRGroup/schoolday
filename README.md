# schoolday

> 育人集团网站主题页面

## 使用方法

``` 命令行（PowerShell、cmd）
# 安装项目依赖库
cnpm i

# 运行本地热重载服务器 localhost:8080
npm run dev

# 打包压缩项目
npm run build

```

目录结构：  
- 源码 ../src/
- 素材 ../src/assets/
- html模板  ../index.html
- 入口文件  ../src/main.js
- 主页面  ../src/App.vue
- 公用数据  ../src/store/index.js
- PC页面  ../src/page
- 移动端页面  ../src/mobile
- 路由表  ../src/router/index.js
- 打包后的静态文件  ../dist
- 项目配置文件  ../config  ../build
